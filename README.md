# Automated AWS EC2 Web Server Deployment Using Terraform

## Project Overview
This project automates the deployment of an AWS EC2 web server using Terraform.

## Technologies Used
- AWS EC2
- Terraform
- Ubuntu Linux
- Apache2
- Security Groups

## Features
- Infrastructure as Code (IaC)
- Automated EC2 provisioning
- Security Group configuration
- Apache web server hosting
- Terraform Variables
- Terraform Outputs

## Project Structure

terraform-ec2-webserver/
│
├── provider.tf
├── variables.tf
├── main.tf
├── outputs.tf
├── userdata.sh
├── README.md
└── screenshots/

## Terraform Commands

terraform init
terraform validate
terraform plan
terraform apply
terraform destroy

## Outcome
Successfully deployed a web server on AWS EC2 using Terraform and hosted a sample webpage.
